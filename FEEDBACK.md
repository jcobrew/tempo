# Feedback Prompts

Use this exact prompt when sending the build to testers:

## Short Version

What did you use Tempo for, what felt useful, what felt awkward, and what should be improved first?

## Better Version

Please reply with:
1. The task you used it for
2. Whether the floating mini timer helped or distracted you
3. Whether the nudges felt useful, neutral, or annoying
4. Anything confusing in the main window or setup flow
5. One thing you would want changed before using it again

## Useful Metadata

If they are willing, also ask:
- macOS version
- Apple Silicon or Intel
- whether notifications were enabled
- whether they tried strict mode
- whether they used music
